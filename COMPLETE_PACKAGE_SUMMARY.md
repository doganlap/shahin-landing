## ğŸ› Common Issues & Solutions

### Issue: Build fails
**Solution:** Clear cache, reinstall dependencies
```powershell
rm -rf node_modules dist
npm install
npm run build
```

### Issue: Database migration fails
**Solution:** Check connection, run one at a time
```powershell
psql $env:DATABASE_URL -f 001_landing_cms.sql
# Verify before running next
psql $env:DATABASE_URL -f 002_sandbox_system.sql
```

### Issue: Sandbox not creating
**Solution:** Check backend logs, verify JWT secret
```powershell
az containerapp logs show --name grc-backend-prod ...
```
